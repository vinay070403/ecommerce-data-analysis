# 📊 Amazon India E-Commerce Sales Analysis

## 📌 Project Overview
Analysis of 1.28 lakh+ Amazon India fashion sales records to uncover revenue trends, category performance, and order insights.

## 📁 Project Structure
```
ecommerce-data-analysis/
│
├── sql/
│   └── analysis.sql
│
└── Amazon-Ecommerce-Analysis.xlsx
```

## 🔍 Key Analyses Performed

### Excel Dashboard
- Category wise Revenue Breakdown
- State wise Sales Performance  
- Order Status Analysis
- KPI Cards — Total Revenue, Total Orders, Cancelled Orders

### SQL Queries
- Total Revenue & Order Count
- Monthly Revenue Trend
- Top Selling Categories
- Order Status Breakdown
- Top 10 States by Revenue
- B2B vs B2C Comparison
- Cancellation Rate by Category

## 📈 Key Insights
- **Kurta** is the top selling category with highest revenue
- **Maharashtra & Karnataka** are top performing states
- **Shipped orders** contribute majority of total revenue
- Cancellation rate analysis helps identify problem categories

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Dashboard, Charts)
- SQL Server (Data Validation & Analysis)
- GitHub (Version Control)

## 📂 Dataset
- Source: Kaggle — Amazon India Fashion Sales Dataset
- Records: 1,28,976 rows
- Fields: Order ID, Date, Status, Category, Amount, State, etc.
